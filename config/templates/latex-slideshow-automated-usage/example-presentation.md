# کد نویسی

## مثال کد جاوا

در زیر یک برنامه ساده جاوا را مشاهده می‌کنید:

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("سلام دنیا!");
        System.out.println("Hello, World!");
    }
}
```

### توضیحات کد

- `public class HelloWorld`: تعریف کلاس عمومی
- `main`: نقطه شروع برنامه
- `System.out.println`: چاپ خروجی

## کد پایتون

```python
def fibonacci(n):
    """محاسبه دنباله فیبوناچی"""
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# نمایش ۱۰ عدد اول
for i in range(10):
    print(f"F({i}) = {fibonacci(i)}")
```
