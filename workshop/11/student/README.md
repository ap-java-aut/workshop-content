توی کارگاه یازدهم قراره به موضوع سورت و سرچ بپردازیم.

تا حالا احتمالاً بدون اینکه متوجه بشین، از الگوریتم‌های مرتب‌سازی و جستجو توی برنامه‌هاتون استفاده کردین. بعد از گذروندن درس «برنامه‌سازی پیشرفته»، توی درس «ساختمان داده‌ها و الگوریتم‌ها» به‌طور مفصل با این مباحث آشنا می‌شین. اما توی این هفته می‌خوایم یه مرور کلی روی الگوریتم‌های مرتب‌سازی مثل Bubble Sort، ‏Quick Sort‏‏ و جستجو مثل Linear Search و Binary Search داشته باشیم.

دو داکیومنت برای این هفته داریم: داکیومنت **[«مقدمه‌ای بر الگوریتم‌های سرچ»](search.pdf)** و داکیومنت **[«مقدمه‌ای بر الگوریتم‌های سورت»](sort.pdf)**. این داک‌ها بهتون کمک می‌کنن تا با مفاهیم پایه و پیاده‌سازی این الگوریتم‌ها توی جاوا آشنا بشین.
