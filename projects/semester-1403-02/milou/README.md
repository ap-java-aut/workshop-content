پروژهٔ نهایی‌تون، برای جمع‌بندی تمام چیزهاییه که توی این ترم یاد گرفتین. ممکنه حین انجام این پروژه، کلی سوال براتون پیش بیاد، همگی اون‌ها رو می‌تونید از تدریس‌یارهاتون بپرسین، حتی اگر سوالات ساده‌ای به نظر برسن.

توی این پروژه، شما باید از دیتابیس استفاده کنید. دقت کنید که فایل‌های SQL لازم برای ساخت دیتابیس‌تون باید توی ریپوی گیت‌تون‌ موجود باشه. 

مثل همیشه سعی کنید که یک ریپوی منظم، با کامیت‌های مناسب و خوب ایجاد کنید و دایرکتوری‌ها و فایل‌های نامربوط رو توی اون کامیت نکنید. همچنین، لطفا به هیچ وجه اطلاعات دیتابیس‌تون مثل URL دیتابیس و پسوردش رو توی ریپوتون ننویسید. **ریپوتون تا انتهای ددلاین پروژه باید پرایوت باشه و بعد ددلاین لازمه اون رو پابلیک کنید**.

دقت کنید که به خاطر زمان‌بندی اعلام نمرات، امکان تمدید این پروژه وجود نداره. حتما برای انجام اون برنامه‌ریزی مناسب داشته باشین. این پروژه ارائهٔ آنلاین داره و لازمه بعد از اتمام ددلاین، اون رو به تدریس‌یارهاتون ارائه بدین.
