توی این تمرین، می‌خوایم یه سیستم ساده درست کنیم که کاربرها بتونن توش با username و password دلخواه خودشون ثبت‌نام کنن، و بعد‌تر هم با همون username و password واردش بشن. توی این تمرین، باید اطلاعات کاربران رو توی دیتابیس ذخیره کنید تا اگر برنامه‌تون بسته شد و دوباره باز شد، کاربران قدیمی همچنان بتونن به حساب کاربری‌شون وارد بشن.

# رابط کاربری

با شروع برنامه، به کاربر دو گزینه بدین:

```
[L]ogin, [S]ign up: 
```

در صورتی که کاربر، حرف `L` یا عبارت ‍‍`Login` رو وارد کرد، باید وارد فرآیند ورود به حساب کاربری بشه و در صورتی که حرف `S` یا عبارت `Sign up` رو وارد کرد، باید وارد فرآیند ثبت نام بشه.

بعد از اتمام فرآیند ثبت‌نام یا ورود به سیستم، از برنامه خارج بشید!

# فرآیند ثبت‌نام

وقتی کاربر خواست ثبت‌نام کنه، ازش اطلاعات زیر رو بخواین:

```
First Name:
Last Name:
Age:
Email: 
Password: 
```

از ایمیل کاربران‌تون به عنوان username استفاده کنید. لازمه حالات زیر رو چک کنید:

- اگر ایمیل کاربر تکراری بود، بهش خطای `An account with this email already exists` رو بدین.
- اگر رمز عبور کاربر کمتر از ۸ کاراکتر داشت،`Weak password` بدین.

بعد از این که کاربر اطلاعات خودش رو وارد کرد، اون‌ها رو توی دیتابیس ذخیره کنید تا بعدا بتونه باهاشون به سیستم وارد بشه.

# فرآیند ورود به سیستم

اگر کاربر خواست به سیستم وارد بشه، ازش ایمیل و پسوردش رو بخواین:

```
Email: 
Password: 
```

چک کنید که اطلاعات کاربر درست باشه. اگر نبود، بهش خطای لازم رو خروجی بدین. وقتی کاربر وارد شد، یک پیام خوش‌آمدگویی به همراه اسمش بهش نشون بدین:

```
Welcome, Gholi Mohammadi!
```

# چیزی که باید آپلود کنید

یه **ریپوی پرایوت** توی گیت‌هاب درست کنید. ریپوتون باید حاوی فایل‌های sqlای که برای ایجاد جداول دیتابیسی‌تون نوشتید هم باشه تا تدریس‌یارها بتونن به درستی دیتابیس‌تون رو بالا بیارن. دقت کنید که دایرکتوری‌ها و فایل‌های نامربوط به کدتون نباید توی ریپو وجود داشته باشه.

نهایتا، توی یه فایل txt، لینک ریپوتون رو بنویسید و اون رو آپلود کنید. بعد از اتمام ددلاین تمرین می‌تونید ریپوتون رو پابلیک کنید.
